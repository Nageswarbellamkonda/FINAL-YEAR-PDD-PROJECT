## 2. Enforced Government Service Formatting
Every answer generated by the Assistant page now strictly incorporates the following details:
1. **Short Explanation**: Direct answer describing the feature/service.
2. **Available Services**: Factual bullet list of sub-components.
3. **Recommended Option**: Bold/italic highlight suggesting the best portal mechanism.
4. **Related Services**: Bracketed list of relative portals (e.g. `• [Track Case], [My Complaints]`).
5. **Next Suggested Step**: Clean block prefixed with `👉` directing immediate citizen next moves.
6. **Action Buttons**: Directly rendered underneath supporting links or click-to-call direct dialers.
